================================================================================
                              TECHNICAL SPECIFICATIONS
================================================================================

Technology Stack:
- Frontend: HTML5, CSS3, JavaScript, FontAwesome
- Backend: PHP 7.4+, MySQL 8.0+
- Security: Password hashing (bcrypt), SQL injection prevention
- File Handling: PDF processing, image uploads
- Email: PHPMailer integration
- Database: PDO, MySQL with proper indexing

Key Files Structure:
- Authentication: Login&Signup.php, session_manager.php
- Dashboards: CandidateDashboard.php, CompanyDashboard.php
- Job Management: JobPost.php, job_posting_handler.php
- Assessment: CreateExam.php, attendexam.php, ExamReview.php
- AI Features: AIMatching.php, CvChecker.php
- Interview: Interview.php, interview_schedule.php
- Notifications: notification_system.js, email handlers
